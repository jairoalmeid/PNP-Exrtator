# PNP-Exrtator
Extração de arquivo .csv da Plataforma Nilo Peçanha utilizando a ferramenta Puppeteer

## Requisitos
*NodeJS*
Instalação do NodeJS - https://nodejs.org/pt

*Instalação do Puppeteer*
`npm install pupperteer`
`npm install @puppeteer/replay`
